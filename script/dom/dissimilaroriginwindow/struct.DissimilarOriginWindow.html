Represents a dissimilar-origin Window that exists in another script thread.

Since the Window is in a different script thread, we cannot access it directly, but some of its accessors (for example window.parent) still need to function.

In browsingcontext.rs, we create a custom window proxy for these windows, that throws security exceptions for most accessors. This is not a replacement for XOWs, but provides belt-and-braces security.
