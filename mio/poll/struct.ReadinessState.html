struct ReadinessState(usize);

Tracks all state for a single ReadinessNode. The state is packed into a usize variable from low to high bit as follows:

4 bits: Registration current readiness 4 bits: Registration interest 4 bits: Poll options 2 bits: Token position currently being read from by poll 2 bits: Token position last written to by update 1 bit: Queued flag, set when node is being pushed into MPSC queue. 1 bit: Dropped flag, set when all Registration handles have been dropped.
