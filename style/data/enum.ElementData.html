Enum style::data::ElementData [−] [src]

pub enum ElementData {
    Initial(Option<ElementStyles>),
    Restyle(RestyleData),
    Persistent(ElementStyles),
}

Style system data associated with a node.

In Gecko, this hangs directly off a node, but is dropped when the frame takes ownership of the computed style data.

In Servo, this is embedded inside of layout data, which itself hangs directly off the node. Servo does not currently implement ownership transfer of the computed style data to the frame.

In both cases, it is wrapped inside an AtomicRefCell to ensure thread safety.
