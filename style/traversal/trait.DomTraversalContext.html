unsafe fn ensure_element_data(element: &N::ConcreteElement)
                              -> &AtomicRefCell<ElementData>

Ensures the existence of the ElementData, and returns it. This can't live on TNode because of the trait-based separation between Servo's script and layout crates.

This is only safe to call in top-down traversal before processing the children of |element|.
