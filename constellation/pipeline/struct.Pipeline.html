A uniquely-identifiable pipeline of script thread, layout thread, and paint thread.

Fields

id: PipelineId frame_id: FrameId

The ID of the frame that contains this Pipeline.

parent_info: Option<(PipelineId, FrameType)> script_chan: Rc<ScriptChan> layout_chan: IpcSender<LayoutControlMsg>

A channel to layout, for performing reflows and shutdown.

compositor_proxy: Box<CompositorProxy + 'static + Send>

A channel to the compositor.

url: ServoUrl

URL corresponding to the most recently-loaded page.

title: Option<String>

The title of the most recently-loaded page.

size: Option<TypedSize2D<f32, PagePx>> running_animations: bool

Whether this pipeline is currently running animations. Pipelines that are running animations cause composites to be continually scheduled.

children: Vec<FrameId> is_private: bool

Whether this pipeline is considered distinct from public pipelines.

visible: bool

Whether this pipeline should be treated as visible for the purposes of scheduling and resource management.
