    pub scrolling: ScrollingState,

    pub parent: Option<ScrollLayerId>,
    pub children: Vec<ScrollLayerId>,
    pub node_type: NodeType,

Contains scrolling and transform information stacking contexts.

Fields

scrolling: ScrollingState

Manages scrolling offset, overscroll state etc.

content_size: LayerSize

Size of the content inside the scroll region (in logical pixels)

local_viewport_rect: LayerRect

Viewing rectangle in the coordinate system of the parent reference frame.

local_clip_rect: LayerRect

Clip rect of this node - typically the same as viewport rect, except in overscroll cases.

combined_local_viewport_rect: LayerRect

Viewport rectangle clipped against parent layer(s) viewport rectangles. This is in the coordinate system of the parent reference frame.

world_viewport_transform: LayerToWorldTransform

World transform for the viewport rect itself. This is the parent reference frame transformation plus the scrolling offsets provided by the nodes in between the reference frame and this node.

world_content_transform: LayerToWorldTransform

World transform for content transformed by this node.

pipeline_id: PipelineId

Pipeline that this layer belongs to

parent: Option<ScrollLayerId>

Parent layer. If this is None, we are the root node.

children: Vec<ScrollLayerId>

Child layers

node_type: NodeType

Whether or not this node is a reference frame.
